# Guides

機能別のワークフロー解説。Getting Started を一通り終えた後、特定の作業を進めるための「2 周目」。

> **TBD**: 本章は未執筆。

## 想定読者

- Getting Started で全体感を掴んだ方
- 自分のプロジェクトに another-mood を導入する方

## このガイドで触れる内容（予定）

- **another-mood とは何か** — コンセプト・できること・できないこと
- **プロジェクト構成** — `contents/` と `definition/` の役割分担
- **データのモデリング** — schema の設計指針
- **コンテンツの書き方** — YAML / Markdown でのデータ記述
- **クエリの書き方** — データの結合・フィルタ・並び替え
- **テンプレートの書き方** — Jinja2 ベースのテンプレート設計
- **ビルドと watch** — `mood build` / `mood watch` の使い分け
