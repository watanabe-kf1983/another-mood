# another-mood ユーザガイド

> **Note**: 本ガイドは執筆中。空欄や TBD 表記が多く残っている段階。

## 構成

3 層構成で、目的に応じて読み進める:

1. **[Getting Started](getting-started.md)** — 1 周目。インストールから最小例のビルドまで一気通貫で動かす
2. **[Guides](guides.md)** — 2 周目。各機能の使い方をワークフロー単位で解説
3. **[Reference](reference/index.md)** — 引く用。CLI コマンド・スキーマ・クエリ・テンプレート等の機能別リファレンス

初めての方は Getting Started から順に。具体的なやり方を調べたい場合は Guides、構文や全オプションを確認したい場合は Reference を参照。
